# AI_Family
This project presents a novel Multi-Agent Framework that bridges the gap between high-fidelity building physics and human cognitive modeling. By coupling the EnergyPlus simulation engine with Large Language Model (LLM) agents (specifically Qwen-Plus).
本项目提出了一种创新的多智能体框架，旨在弥合高保真建筑物理仿真与人类认知模型之间的鸿沟 。通过将 EnergyPlus 仿真引擎与基于 大语言模型 (LLM) 的智能体（如 Qwen-Plus）耦合，系统从传统的数值控制转向了基于语义感知的管理范式 。
